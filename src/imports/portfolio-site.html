Build me a complete single-page portfolio website for a video editor and AI creator. The structure mirrors the Jorge Framer template exactly — sticky nav, hero, marquee, selected works, services, about, footer CTA — but the aesthetic is a fusion of two systems: cinematic film-noir editorial meets minimalist tech-noir glassmorphism. Single HTML file, no frameworks, no libraries, pure HTML + CSS + vanilla JS. DESIGN SYSTEM The entire page lives on a pure black (#000000) background. Primary text is pure white (#ffffff). Secondary/muted text is white at 40% opacity (rgba(255,255,255,0.4)). The only accent is amber (#e8a945) used sparingly — hero outline text, section labels, hover states, the blinking availability dot. No other colors exist. All cards and containers use glassmorphism: background rgba(255,255,255,0.03), backdrop-filter blur(20px), border 1px solid rgba(255,255,255,0.08). Hover state on glass cards elevates to rgba(255,255,255,0.06) with border rgba(255,255,255,0.15). All interactive elements have a subtle white glow on hover using box-shadow: 0 0 24px rgba(255,255,255,0.06). Film grain texture overlays the entire page via a body::before with SVG feTurbulence noise at opacity 0.025. Border radius is 12px on all cards and containers — sharp but refined. Spacing follows strict 8px grid throughout. TYPOGRAPHY Import Clash Display from Fontshare: @import url('https://api.fontshare.com/v2/css?f[]=clash-display@700,600,500&display=swap'). SCROLL REVEAL IntersectionObserver watches all elements with class .reveal. On intersection: remove translateY(40px) opacity-0, add translateY(0) opacity-1, transition 0.8s cubic-bezier(0.23,1,0.32,1). Stagger children using dataset delay attributes set in HTML (data-delay="0.1" through 0.5s), applied as animation-delay in the observer callback.
